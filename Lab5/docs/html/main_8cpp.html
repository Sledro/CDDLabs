void makeProducers ( std::shared_ptr< SafeBuffer >  safeBuff)

The producer generates random characters from ‘a’ to ‘z’ at random intervals (between 0 and 1 second in length).

It adds these to a thread safe buffer that has a finite holding capacity of N characters. It generates a preset number of characters (determined at runtime) and when it has finished it add an ‘X’ character to the buffer and exits.
